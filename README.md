# Rps-react-tf

To run offline -
1) Clone the repo
2) Extract it
3) Install NodeJs (https://nodejs.org/en/)
4) Open terminal/powershell on the window
5) Type "yarn prep" to load dependencies
6) tyoe "yarn start" to start a live server.
